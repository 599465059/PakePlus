================================================================================
                              网页端文件功能说明
================================================================================

一、文件结构
--------------------------------------------------------------------------------

V2.0.8/
├── index.html          # 主页面入口
├── css/
│   └── main.css        # 样式文件
└── js/
    ├── app.js          # 主应用程序
    ├── controls.js     # 控件组件
    ├── device.js       # WebSocket设备管理
    ├── dataParser.js   # 数据解析器
    ├── eventBus.js     # 事件总线
    ├── storage.js      # 本地存储
    └── fileSystem.js   # 文件系统（Tauri）

================================================================================

二、各文件功能详解
================================================================================

【index.html】
    作用: 主页面入口文件
    功能: 
        - 定义页面结构
        - 引入 CSS 和 JS 文件
        - 包含 SVG 图标定义

--------------------------------------------------------------------------------
【css/main.css】
    作用: 样式文件
    功能:
        - 定义页面布局和配色
        - 控件样式
        - 响应式设计

--------------------------------------------------------------------------------
【js/app.js】
    作用: 主应用程序（核心文件）
    功能:
        - 初始化应用
        - 渲染界面（前面板/后面板/配置管理/数据查看）
        - 处理用户交互
        - 管理配置保存/加载
        - 发送/接收 WebSocket 消息
        - 页面关闭时发送 exit 消息

--------------------------------------------------------------------------------
【js/controls.js】
    作用: 控件组件库
    功能:
        - 定义各种 LabVIEW 风格控件
        - 数值控件（旋钮、滑块、数值输入）
        - 字符串控件
        - 布尔控件（按钮、LED、开关）
        - 波形图表
        - 枚举控件（下拉框）

--------------------------------------------------------------------------------
【js/device.js】
    作用: WebSocket 设备管理器
    功能:
        - 连接/断开 WebSocket
        - 自动重连机制
        - 接收 LabVIEW 数据
        - 发送数据到 LabVIEW
        - 维护通道数据缓存
        - 更新连接状态UI

--------------------------------------------------------------------------------
【js/dataParser.js】
    - 作用: LabVIEW 数据解析器
    - 功能:
        - 解析多种格式的 JSON 数据
        - 提取通道名称、值、单位
        - 支持简单键值对、带单位对象、数组等格式

--------------------------------------------------------------------------------
【js/eventBus.js】
    作用: 事件总线
    功能:
        - 发布/订阅模式
        - 组件间通信
        - 定义事件: device:data, control:change, config:saved 等

--------------------------------------------------------------------------------
【js/storage.js】
    作用: 本地存储模块
    功能:
        - 封装 localStorage API
        - 保存/读取 JSON 数据
        - 用于浏览器模式下的配置存储

--------------------------------------------------------------------------------
【js/fileSystem.js】
    作用: 文件系统模块（Tauri 专用）
    功能:
        - 使用 Tauri fs API 读写文件
        - 保存配置到应用目录 jsset/ 文件夹
        - 管理 index.json 索引文件
        - 支持配置删除、设为默认等操作
